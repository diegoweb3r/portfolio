<h1 align="center"> 🖥️ DIEGO WEBER - Portfólio </h1> <h2 align="center"> 📖 Descrição </h2> <p align="justify"> O <strong>Portfólio Diego Weber</strong> é uma aplicação web desenvolvida para apresentar de forma profissional e impactante a trajetória, habilidades e trabalhos de um desenvolvedor de softwares voltado ao front-end. O projeto tem como foco destacar a as habilidades de construção de um website moderno, fluído e acessível. pensado na estética, funcionalidade e usabilidade. </p> <p align="justify"> Com atenção especial à estética visual, experiência do usuário e SEO, foi utilizando design responsivo e uma arquitetura de CSS modular que garante escalabilidade e fácil manutenção. </p>

### 🛠 Tecnologias Utilizadas  
- **Estruturação:** HTML5 Semântico para acessibilidade e SEO.
- **Estilização:** CSS3 Moderno utilizando Metodologia Modular e Custom Properties (Variáveis CSS).
- **Layout:** CSS Grid e Flexbox para responsividade intrínseca (sem dependência excessiva de Media Queries).
- **Tipografia:** Tipografia moderna para uma interface contemporânea e legível.
- **Versionamento:** Git & GitHub para controle de fluxo e documentação.
